# 🎬 Cinemania

Cinemania is a responsive movie discovery web application that allows users to explore trending movies, search for specific films, watch trailers, and manage a personal movie library.

The project focuses on building a modern front-end application using vanilla JavaScript and modular architecture while integrating external movie APIs.

## 🚀 Features

- Browse trending movies
- Search movies by title
- Watch movie trailers
- View detailed movie information
- Add movies to personal library
- Responsive design for mobile, tablet, and desktop
- Modern UI based on Figma design

## 🛠️ Technologies Used

- HTML5
- SCSS / CSS3
- JavaScript (ES6+)
- REST API integration
- Parcel / Vite bundler
- BEM methodology
